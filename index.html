Express

var app = express.createServer();

app.get('/', function(req, res){
  res.send('Hello World');
});

app.listen(3000);

主な特徴

  • 堅牢なルーティング
  • リダイレクト・ヘルパー
  • 動的なビューヘルパー
  • アプリケーションレベルで設定可能なビュー・オプション
  • コンテントネゴシエーション
  • アプリケーションのマウント
  • 高いパフォーマンスへのフォーカス
  • ビューの描画とパーシャルのサポート
  • 環境にもとづく構成
  • セッションにもとづくフラッシュ通知
  • Connect上に構築
  • Executableによる素早いアプリケーション生成
  • 広範なテスト範囲

サードパーティモジュール

以下のモジュール群はExpress公認のものです。

  • express-resource - 優秀なルーティング機構を提供
  • express-messages - flash message notification rendering
  • express-configure - 非同期な環境設定をサポート (load settings from redis etc)
  • express-namespace - 名前空間によるルーティングをサポート
